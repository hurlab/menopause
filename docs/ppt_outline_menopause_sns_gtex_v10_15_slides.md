### Slide 2 — Kenichi’s hypothesis + what we need from human data
**Title:** Working hypothesis and requested human evidence (Kenichi)

**On-slide bullets:**
- Hypothesis: ↓ central estrogen → ↓ D2 signaling → ↑ sympathetic outflow to adipose → ↑ lipolysis → insulin resistance.
- Mouse evidence: OVX shows ↓ dopamine release + ↑ SNS activity; THKO/AAKO/bromocriptine improve OVX IR.
- Desired human support: adipose transcription consistent with ↑ adrenergic signaling and/or ↑ lipolysis program in postmenopause.
- Caveats (Kenichi): GTEx is baseline/unstimulated; acute SNS markers may be weak; chronic SNS can cause catecholamine resistance (e.g., ADRB3 down).

**Speaker notes (framing):**
- We explicitly test “acute SNS response” vs “program-level/chronic remodeling” readouts.

### Slide 15 — Synthesis + recommended manuscript framing + next steps
**Title:** What we can claim now + recommended next steps

**Most defensible claims (GTEx baseline adipose):**
- Acute SNS immediate-early markers are not robustly upregulated at baseline.
- Subcutaneous shows significant downshift in lipolysis_core and thermogenesis_program activity across menopause proxy groups.
- Visceral shows thermogenesis_program downshift; lipolysis_core not strong.
- Menopause signature scores increase across groups (stronger subcutaneous; weaker visceral).

**How to frame with Kenichi caveats:**
- Baseline expression may reflect chronic adaptation/catecholamine resistance rather than acute SNS activation.
- Lack of acute marker upregulation does not rule out increased sympathetic outflow.

**Immediate deliverables (1–2 weeks):**
- Finalize a two-depot figure panel set (subcutaneous + visceral) with conservative caption language.
- Refine curated signature list to improve adipocyte/biological interpretability: `references/menopause_signature_curated.tsv`
- Send short update package with both depots (reports + figure candidates).
